## Tutorial
Use WASD keys to movement character,
QE keys to switch weapeons
and click with in direction you want to attack

````
MOVEMENT PLAYER
   [W]
[A][S][D]

SWITCH WEAPEONS
[Q]  [E]

ATTACK
MOUSE LEFT -> Attack
click in direction you want to attack
````
